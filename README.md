# CY_repo featuring Web Development Concepts, Project

This project demonstrates the fundamental concepts of web page development, covering the core building blocks of HTML web page structure e.g `<DOCTYPE html>`, `<html>`, `<head>`, and `<body>` and CSS structuring and styling. The project serves as a practical implementation of structuring and styling web content.

## Project Structure
Here are the main files that make up this repository:
- `index.html` - The main structure of the web page, containing all the `HTML` elements and content.
- `style.css` - The external stylesheet used to design, layout, and style the web page.

## Features & Concepts Implemented

### HTML Fundamentals
- **Document Structure**: Proper usage of `DOCTYPE html`, `<html>`, `<head>`, and `<body>`.
- **Metadata**: Adding page titles inside the `<head>` tag.
- **Core Syntax**: Practical application of key terminologies like elements, tags, attributes, opening tags, and closing tags.
- Typography: Structuring content using headings `(<h1>, <h2>)` and paragraphs `(<p>)`.
- **Links**: Hyperlinks using the `<a>` tag with the `href` attribute.
- **Images**: using the `<img>` tag  which does not have a closing tag with `src` and `alt` attributes.
- **Lists**: Creating organised content using unordered lists `(<ul>)`.
- **Layout & Styling Hooks**: Grouping content with `<div>` tags and highlighting inline text with `<span>` tags.

### CSS Styling & Architecture
- **Integration**: Connecting CSS directly to HTML using a style sheet.
- **CSS Rule Anatomy**: Implementation of standard CSS selectors and declaration block structures.
- **Selectors**: Targeting elements precisely using:
	- Element selectors (e.g `p`, `div`)
	- Class selectors (e.g `.classname`)
	- ID selectors (e.g `#idname`)
- **Typography Styling**: Changing text color, font sizes, and font families.
- **Theming**: Modifying background colors for both individual elements and the entire page.
- **The Box Model**: Applying borders, margins, and padding to control spacing.
- **Alignment**: Centering text and images for a balanced layout.
